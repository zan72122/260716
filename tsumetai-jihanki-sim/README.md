# つめた〜い じはんきシミュレーター

日本の冷飲料自動販売機の「なかみ」を物理シミュレーションで狂気的に再現した 3D 観察ゲーム。
実在機 **富士電機 FR30A6R40TK-FOP**（30セレクション・36押ボタン・W1185×H1830×D731mm）の
公開仕様・特許文献・業界資料をもとに、前面レイアウトから内部機構までを実機準拠でモデル化しています。
iPhone / iPad の縦画面・横画面どちらでも遊べます。ビルド不要・静的サーバーだけで動きます。

```
python3 -m http.server   # リポジトリルートで実行
# → http://localhost:8000/tsumetai-jihanki-sim/
```

## 実機準拠ポイント

- **前面**: サンプル展示室 3段×12本 (計36サンプル・PET上段/太缶中段/細缶下段)、各サンプル直下に
  価格プレート+押ボタン (購入可=青点灯 / 売切=赤)、青い「つめた〜い」帯、
  右縦帯に 7セグ金額表示・縦型硬貨投入口・千円札口(緑LED)・回転式返却レバー・ICリーダー・
  つり銭切れ/お札中止ランプ、下部に横長「お取り出し口」フラッパー、右下に蓋付きつり銭口。
  統一ステッカー・住所表示ステッカー・新500円対応・銘板・黒い蹴込みルーバーまで再現
- **庫内**: 3室 × 上下段ラック × 前後5コラム = **30コラム**の五重サーペンタイン。
  ほぼ垂直の細かい蛇行チャンネルを商品が転がり降り、段違いに並んだ**2ピン式ベンドメカ**が1本ずつ払出し。
  落下した商品はデフレクタで加速され**搬出扉(断熱フラップ)**を押し開けて取出口へ「ガコンッ」
- **コインメック**: 硬貨は実寸の直径・質量の剛体。傾斜レールの**径ゲート**で物理選別され、
  **100/500円は保留シャッターに一時保留 (エスクロー)** — 購入確定でチューブ/金庫へ、
  返却レバーを回すと**投入した現物がそのまま**つり銭口に落ちてくる (実機と同じ)
- **千円札**: ビルバリデータが搬送・判定。釣銭13枚相当を確保できないときは
  「お札中止」ランプが点いて受け付けず、吐き出される (実機の受入ルール)
- **釣銭**: 貪欲法で計算し、チューブ底のエジェクタが1枚ずつ蹴り出して物理落下。
  売切・つり銭切れランプはすべて在庫実数と連動
- **モード**: 🩻X線透視 / ⏸・¼・⅒スローモーション / 🔧店員モード
  (扉開閉 → トップトレーに補充すると商品が転がって空きコラムに落ちる・釣銭補充・売上回収)

## 実装

- `js/physics.js` — 自作 2D 平面物理 (固定 1/480s ステップ・逐次インパルス・スリープ・
  センサー線・レイヤーバケツ)。商品約160本+硬貨をモバイルでも実時間で解く
- `js/coin-mech.js` — コインメック+ビルバリ。機構の 3D 表示は物理コライダから自動生成し、
  見た目と挙動が乖離しない
- `js/rack.js` — 30コラム生成・ベンドメカ・販売検知 (不発時は自動リトライ→全額返金)・トレー補充
- `js/vending-state.js` — 36ボタン→30コラムのセレクション解決・エスクロー・釣銭・売上
- 効果音は全て WebAudio 合成。スロー時は音程・余韻も引き伸ばされます
