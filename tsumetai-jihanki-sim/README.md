# つめた〜い じはんきシミュレーター

日本の冷飲料自動販売機の「なかみ」を物理シミュレーションで狂気的に再現した 3D 観察ゲーム。
iPhone / iPad の縦画面・横画面どちらでも遊べます。ビルド不要・静的サーバーだけで動きます。

```
python3 -m http.server   # リポジトリルートで実行
# → http://localhost:8000/tsumetai-jihanki-sim/
```

## あそびかた

- **おさいふのコインをタップ** → 投入口から硬貨が入り、実寸の直径 (10円23.5mm / 50円21.0mm / 100円22.6mm / 500円26.5mm) による**径ゲート選別レール**を転がって金種別の釣銭チューブへ振り分けられる
- 金額が足りるとボタンが点灯 → **ボタンをタップ**でサーペンタインラック下部の**2ピン式ベンドメック**が1本だけ払い出し、ジグザグ棚を転がり落ちてフラップを「ガコンッ」と押し開ける
- **🩻 とうしモード**: 筐体が透けて、コインの選別・商品の落下・釣銭の払出しがぜんぶ見える
- **⏸ / ¼ / ⅒**: 一時停止・スローモーションでじっくり観察
- **🔧 てんいんモード**: 扉を開けて商品補充・釣銭チューブ補充・コイン金庫の売上回収
- 返却レバーで全額返金。釣銭切れ・売切ランプも実数と連動

## しくみ (実装)

- `js/physics.js` — 自作の 2D 平面物理 (固定 1/480s ステップ・逐次インパルス・スリープ・センサー線)。硬貨はメック面、商品は各コラム面の 2D 剛体として解かれ、3D は補間表示
- `js/coin-mech.js` — コインメック: 投入口 → 返却ゲート → 傾斜選別レール (径ゲート) → 検銭センサー → 振分フリッパー → 釣銭チューブ / 金庫。返却・払出しもすべて物理落下
- `js/rack.js` — サーペンタインコラム生成・ベンドメック・販売検知 (出ない場合は自動リトライ→返金)
- `js/vending-state.js` — credit / 貪欲法の釣銭計算 / 売切・釣銭切れ判定 / 売上
- 機構の 3D 表示は物理コライダから自動生成しており、見た目と挙動が乖離しません
- 効果音はすべて WebAudio 合成 (音声ファイルなし)。スロー時は音程・余韻も引き伸ばされます
