# にじいろカート アイランドGP 🏎️🌈

4歳児向けにデザインした、マリオカート ワールド風の 3D カートレースゲームです。
iPhone / iPad の **縦画面・横画面どちらでも** 遊べます。ブラウザだけで動作し、外部アセットは一切不要です。

![対象](https://img.shields.io/badge/%E5%AF%BE%E8%B1%A1-4%E6%AD%B3%E3%81%8B%E3%82%89-ff9f1c)

## あそびかた

1. すきな どうぶつを えらぶ（くまの モカ / うさぎの ミミ / ぺんぎんの ペン / ねこの トラ）
2. **スタート！** をタップ
3. アクセルは じどう。ハンドルそうさだけで OK！
   - がめんの **← → ボタン** をおす
   - または がめんを **よこに なぞる**（ドラッグ）
   - PC なら やじるしキー / A・D キー（スペースで アイテム）
4. スタートラインを 3 かい くぐったら ゴール！

### コースのおたのしみ

- 🪙 **コイン** … とると ちょっとだけ スピードアップ
- 🎁 **はてなボックス** … ロケットダッシュ / キラキラスター / コインマグネット がでるよ（ほうっておいても じどうで つかってくれる）
- 🔶 **ダッシュパネル** … ふむと ビューン！
- 🌈 **にじのはし**・🎡 **かんらんしゃ**・🎈 **ききゅう**・⛵ **ヨット** も みつけてね

なにもそうさしなくても、ゆるやかに コースのまんなかへ もどるアシストつき。かべで コースのそとに おちることはありません。

## 実行方法

ビルド不要の静的サイトです。このフォルダを任意の静的サーバーで配信してください。

```bash
cd kart-world-3d
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

GitHub Pages や任意のホスティングにそのまま置いても動きます（`file://` 直開きは ES Modules の制約で不可）。

## 技術ハイライト（Switch 級の見た目を Web で目指した工夫）

| 項目 | 内容 |
|---|---|
| レンダリング | Three.js r160 / ACES Filmic トーンマップ / sRGB / PCF ソフトシャドウ（プレイヤー追従の 2048px シャドウカメラ） |
| ライティング | 太陽光 + 半球光 + トゥーン用 4 段グラデーションマップで任天堂風のセルルック |
| 空 | シェーダーによるグラデーション大気 + 太陽ディスク + 2 層 fbm ノイズの流れる雲 |
| 海 | 頂点波 + きらめきノイズ + 岸辺に寄せる泡リングのカスタムシェーダー |
| 地形 | value-noise fbm の起伏島。道路スプラインへ自動ブレンドして滑らかに接続、砂浜→草原→岩肌を頂点カラーで表現 |
| コース | 閉じた Catmull-Rom スプラインから道路・赤白縁石・白線・黄色破線・土手をリボン生成。橋区間は自動検出して虹の路面 + 欄干 + 橋脚を構築 |
| 植生 | InstancedMesh の木・ヤシ・花・岩 400 個超。葉は頂点シェーダー注入でそよ風に揺れる |
| 演出 | ドリフト火花・土けむり・ブースト炎・スター星屑・紙吹雪（すべて GPU インスタンシング）、ブースト時の集中線オーバーレイ + FOV キック |
| キャラ | Canvas 描画のにっこりフェイステクスチャ + トゥーンシェーディングの動物ドライバー。ハンドル連動の首ふり・車体ロールつき |
| 音 | WebAudio 完全生成：ループ BGM（メロディ+ベース+ハット）、速度連動エンジン音、コイン・ダッシュ・ファンファーレ等の効果音 |
| モバイル | safe-area 対応 / 縦横レイアウト自動切替 / devicePixelRatio 上限 2 / スクロール・ピンチ抑止 / タップでオーディオアンロック |

## 4歳児向けの設計

- アクセル自動・ブレーキなし。そうさは「まがる」だけ
- 手を離すとコース中央へ弱いオートアシスト、見えない壁で場外なし
- CPU はやさしめ + ラバーバンドで「がんばれば 1 い」になる難度
- 文字はすべて ひらがな・カタカナ、大きな UI と大きなあたり判定
- アイテムは押さなくても自動発動、失敗ペナルティなし

## フォルダ構成

```
kart-world-3d/
├── index.html      # UI・HUD・スタイル・importmap
├── lib/
│   ├── three.module.min.js   # Three.js r160（同梱）
│   └── THREE_LICENSE
└── src/
    ├── main.js     # 起動・カメラ・入力・UI・ループ
    ├── world.js    # 空・海・島・コース・草木・ギミック生成
    ├── kart.js     # カートと動物ドライバー
    ├── race.js     # アーケード物理・AI・ラップ・アイテム
    ├── fx.js       # パーティクル・紙吹雪・集中線
    └── audio.js    # WebAudio 生成の BGM / SE
```
