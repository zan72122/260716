# そらのエアライダー 🌟

エアライドレース風の、4歳児向け 3D タッチゲームです。
うさぎのライダー「ピュン」が星のマシンに乗って、空に浮かぶ島のレインボーコースを
ぐるぐる走ります。iPhone / iPad の縦画面・横画面どちらでも遊べます。

> オリジナルキャラクター・オリジナル楽曲によるファンメイドの「エアライド風」ゲームです。
> 任天堂・HAL研究所の作品とは関係ありません。

## あそびかた

| そうさ | うごき |
| --- | --- |
| 画面の左/右をさわる | その方向にまがる |
| どこでも長おし | チャージ(ためる) |
| ゆびをはなす | ブースト!ビューン! |
| ← → キー / スペース | パソコンでも同じそうさ |

- ⭐ をあつめるとカウントが増えて、10こごとにおいわい
- 🌈 リングをくぐると大ブースト
- 🟠 ダッシュパネルでスピードアップ、🔵 ジャンプ台で大ジャンプ
- 失敗やゲームオーバーはありません。ぶつかっても「ぽよん」と鳴るだけ

## 起動方法

ES Modules を使っているため、ローカルの HTTP サーバーから開いてください。

```bash
cd sky-air-rider
python3 -m http.server 8000
# → http://localhost:8000 を Safari / Chrome で開く
```

GitHub Pages などの静的ホスティングにそのまま置いても動きます
(依存はすべて `vendor/` に同梱済み・ビルド不要・完全オフライン動作)。

## 描画のこだわり

- **ソフトトゥーンシェーディング**: 4段グラデーションマップ + リムライトを
  全マテリアルに注入し、やわらかいセル調の質感に
- **ACES フィルミックトーンマッピング + UnrealBloom**: 星やリングがふわっと発光
- **スプライン生成コース**: Catmull-Rom 曲線からバンク(傾き)つきの道路メッシュ、
  虹色のふちどり、キャンディポールを手続き生成
- **生きている世界**: きらめく海のシェーダー、雪冠の山なみ、浮島、気球、
  もこもこ雲、インスタンシングされた木々と花畑
- **エフェクト**: 虹のリボントレイル、GPUパーティクル(収集バースト・紙吹雪)、
  スピードライン、タッチリップル
- **PCFソフトシャドウ**: プレイヤーに追従する太陽光の影カメラ
- **自動品質調整**: フレーム時間を監視して解像度・ブルームを段階調整
  (古い iPad でも快適に)

## 音

すべて WebAudio でリアルタイム合成しています(音源ファイルなし)。
ごきげんな 8 小節ループの BGM、コンボで音階がのぼる収集音、
チャージ上昇音、ブーストのホワイトノイズスイープなど。

## 構成

```
sky-air-rider/
├── index.html        エントリ (importmap / HUD / タイトル)
├── css/style.css     UI スタイル (セーフエリア・縦横対応)
├── js/
│   ├── main.js       ループ / 物理 / カメラ / ポストプロセス / 品質調整
│   ├── world.js      空・海・地形・コース・アイテムの手続き生成
│   ├── rider.js      キャラクターとマシンの造形・アニメーション
│   ├── effects.js    パーティクル / トレイル / 2Dオーバーレイ
│   ├── audio.js      WebAudio 合成 BGM & 効果音
│   ├── input.js      タッチ / キーボード入力
│   ├── ui.js         HUD・メッセージ演出
│   └── lib3d.js      トゥーン質感・ジオメトリ結合などの共通処理
└── vendor/           three.js r170 (MIT) 同梱
```
