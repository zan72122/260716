# テニス フィーバー! 🎾✨

4歳児のための、マリオテニス フィーバー風 3D テニスゲームです。
iPhone / iPad の**縦画面・横画面どちらでも**遊べます。

## あそびかた

1. **タップして はじめる**
2. すきなキャラクターを えらぶ(ミミ 🐰 / トラ 🐱 / ガオ 🦖)
3. ボールが きたら **がめんを タッチ!** それだけ!

- キャラクターは **かってに はしってくれる** ので、タッチのタイミングだけで遊べます
- ラリーを つづけると **フィーバーゲージ** がたまって、まんたんで **フィーバータイム!!** 🌈
  (にじいろの そら・かみふぶき・おんがくも ノリノリに!)
- あいてが とれなかったら **ポイント!** はなびが あがるよ 🎆
- ミスしても「どんまい!」だけ。**ゲームオーバーは ありません**
- あつめた ★スター は ほぞんされます

## 起動方法

ES Modules を使っているため、HTTP サーバー経由で開いてください
(`file://` では動きません)。

```bash
cd tennis-fever-3d
python3 -m http.server 8080
# → http://localhost:8080 を iPhone/iPad の Safari か PC ブラウザで開く
```

GitHub Pages にデプロイすれば、そのまま iPhone/iPad で遊べます。
ホーム画面に追加すると全画面で遊べます(`apple-mobile-web-app-capable` 対応済み)。

## こだわり(Switch 品質の 3D 描写を目指して)

すべて**プロシージャル生成**(外部アセット 0 個・完全オフライン動作)。

### レンダリング
- **ACES Filmic トーンマッピング** + sRGB 出力で、家庭用ゲーム機らしい色乗り
- **PCF ソフトシャドウ**(2048px シャドウマップ、bias 調整済み)
- トゥーンシェーディング + **背面法アウトライン**で、キャラクターは任天堂風の暖かい質感
- コートは **クリアコート付き物理マテリアル**でアクリルコートの艶を再現
- カスタム GLSL の**グラデーション空ドーム**(フィーバー時は虹色バンドが流れる)
- デバイスに応じた**アダプティブ解像度**(重い端末では自動で描画解像度を調整)

### 生きているスタジアム
- **640 体の観客**(インスタンシング + 頂点シェーダーで全員が跳ねる。フィーバー時は大興奮)
- ラリー数を映す**ジャンボトロン(大型ビジョン)**
- はためく**万国旗 44 枚**、ふわふわ漂う**バルーン**、フレア付き**照明塔**
- 外周には トゥーンの丘・木・流れる雲・太陽グロー

### ジューシーな手応え
- ボールの**グロー軌跡**(フィーバー時は虹色スター軌跡)
- ヒット時の**スターバースト + 衝撃波リング + 画面シェイク**
- バウンド時の土煙、紙吹雪の雨、打ち上げ**花火**
- キャラクターは常時アニメ(呼吸・まばたき・耳ゆれ・トコトコ走り・スイング・大よろこび)

### サウンド(WebAudio 完全合成)
- チップチューン BGM ループ(フィーバーで BPM アップ + キラキラアルペジオ)
- ラリーが続くほど**打球音のピッチが上がる**
- 歓声・ファンファーレ・花火・「ぽよん」ミス音まで全部シンセ生成

## 設計メモ

| ファイル | 役割 |
|---|---|
| `index.html` | UI(タイトル / キャラ選択 / HUD)と importmap |
| `css/style.css` | 子ども向けの大きく弾む UI、セーフエリア対応 |
| `js/main.js` | ゲームループ・ボール物理・AI・フィーバー進行 |
| `js/world.js` | スタジアム・空・観客・照明などワールド生成 |
| `js/character.js` | トゥーンキャラ生成 + プロシージャルアニメ |
| `js/effects.js` | パーティクル(全プール化・プレイ中アロケーション 0) |
| `js/audio.js` | WebAudio シンセ(BGM スケジューラ + SFX) |
| `js/vendor/three.module.js` | three.js r160(同梱) |

### 4歳児向けデザイン原則
- 操作は「タッチ」ひとつ。スワイプも長押しもなし
- 当たり判定はとても寛大(半径 2.8m)+ スイートスポットで「ナイス!」ボーナス
- 負けない・怒られない・待たされない(ミス後 1.4 秒で次のサーブ)
- 文字が読めなくても遊べる(絵と音で全部わかる)
- iOS のダブルタップズーム・スクロール・長押しメニューをすべて抑制
