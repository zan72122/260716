# カラーじしゃく すなもよう庭

磁石を動かすと、色のついた磁性砂が遅れて集まり、山・尾・渦・橋・マーブル模様を描く——
4歳児向けのタッチあそびです。iPhone / iPad の縦画面・横画面のどちらでも遊べます。

インストール不要。`index.html` をブラウザ(Safari / Chrome)で開くだけで動きます。
外部ライブラリ・外部通信は一切ありません(音もその場で合成)。

## あそびかた

| そうさ | できること |
| --- | --- |
| 画面にさわる | さわった場所の砂がぴくっと跳ねる(その瞬間に反応) |
| 磁石を1本の指でドラッグ | 砂が遅れて集まってきて、山や尾、渦の模様ができる |
| 磁石をじっと置く | まわりの砂が磁石のほうへ揃い、力線の筋がうかびあがる |
| 速く動かしながら離す | 山の外側がちぎれて、置き跡の模様が残る |
| 何もない所をタップ | 2個目の磁石が上からぽとっと落ちてくる(もう2個あれば、近い磁石がそこへ滑っていく) |
| 磁石2個を近づける | 両側から砂の橋がのびる。近いほど太く、離すと糸になって「ぱちんっ」と切れる |
| 磁石を重ねる | 1個に戻る(きらんと合体) |
| 大きな「ふりふり」ボタン | **磁力が切れて**山がふわっとほどけ、色の波が端から走って平らに。少し置いて磁力が再点灯する |
| 左上の🎨 | 砂の色テーマを切り替え(にじいろ / ゆうやけ / うみ) |
| 右上の🔊 | 音のオン・オフ |

- 磁石は指より大きく、実際のタッチ位置より少し**上**に表示されるので、指で隠れません。
- どこをドラッグしても近くの磁石が指を追いかけるので、小さな子でも迷いません。
- 砂は磁石から離れた場所に置かれるとその場に残ります。平らに戻すのは「ふりふり」だけ。
- しばらくさわらないと、磁石がぷるっと震えて近くの砂を跳ねさせ、あそびに誘います。

## 磁石らしさの表現

- **じわじわ→一気**: 磁力は距離の急坂カーブ。遠くの砂はそよぐだけ、近づくとドサッと吸い付く。
- **力線**: 寝ている砂が磁界の向きに揃い、8本の筋になって見える。2個置くと磁界がつながって曲がる。
- **連鎖吸い**: 捕まった粒が隣の粒を引っ張り、大きくなった山は縁で寝砂を接触で取り込む。
- **吸って運んで置く**: 山の芯ほど磁石に密着して運ばれ、速く動かして離すと外周がちぎれて跡になる。
- **磁力を切る**: ふりふりは磁場オフ。磁石は目を閉じてくすみ、光が消え、砂がほどけ、波のあとに再点灯する。

## 遊ばせ方のヒント(おとな向け)

- ゆっくり動かすと大きな山、速く動かすと途中で砂がちぎれて軌跡の模様が残ります。
- 磁石をぐるぐる円に動かすと、砂山が渦を巻きます。
- 2個の磁石をゆっくり近づけたり離したりすると、橋が太くなったり糸になったりします。切れる瞬間の「ぱちんっ」も楽しい。
- 画面を回転しても描いた模様はそのまま保たれます。
- ホーム画面に追加するとフルスクリーンで遊べます(iOS: 共有 → ホーム画面に追加)。

## フォルダ構成

```
color-magnet-sand-garden/
├── index.html            エントリポイント
├── manifest.webmanifest  ホーム画面追加用
├── css/style.css         UI・タイトル画面のスタイル
├── js/
│   ├── config.js         全調整パラメータと色テーマ
│   ├── audio.js          WebAudio による音の合成
│   ├── sim.js            砂と磁石の物理シミュレーション
│   ├── render.js         Canvas 2D 描画(粒のバッチ描画・磁石スプライト)
│   ├── input.js          タッチ / ポインタ入力
│   └── main.js           起動・ループ・性能自動調整
├── icons/                アイコン(SVG / PNG)
└── docs/DESIGN.md        仕様と実装の対応表・設計メモ
```

## 技術メモ

- Canvas 2D + Vanilla JS(ES5 互換、ビルド不要)
- 粒は並列 TypedArray で管理し、色×明るさごとに 1 パスへまとめて描画
- 粒数は実測 FPS に応じて 1,800〜5,600 の間で自動調整
- 音は WebAudio でノイズ・オシレータから合成(初回タップでアンロック)。
  磁石の速さでさらさら音が変わり、山が大きいほど音が低い。橋の完成はグリッサンド、切断はぱちん
- タッチした瞬間(同フレーム)に粒が反応(pokeAt)— 入力から視覚変化まで 100ms 未満
- 画面回転・リサイズはトレイ矩形基準の正規化座標で再配置し、描いた模様を維持
- 3本以上の同時タッチはタップ扱いしない(手のひら誤タッチ対策)
