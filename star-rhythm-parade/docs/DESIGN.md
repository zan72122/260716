# 設計ドキュメント — ほしぞら リズムパレード

## コンセプト

「リズム天国」シリーズのような **見て・聞いて・まねする** リズムあそびを、
4歳児がひとりで遊べるところまで削ぎ落とした 3D ゲーム。

- **入力はタップひとつ**。画面のどこを叩いてもいい(ボタンの概念がない)
- **合図は必ず「音+動き」の二重化**。お手本の音、振りかぶり、縮むリング、
  飛んでくる星、順番に跳ぶウェーブ — 耳でも目でも「いま!」が分かる
- **失敗が存在しない**。外してもキャラが可愛くよろけるだけで、音楽は止まらない。
  リザルトは必ず ★1 以上で、ポジティブな言葉しか出さない

## 4歳児向けの調整

| 項目 | 値 | 理由 |
| --- | --- | --- |
| ぴったり判定 | ±160ms | 大人向け音ゲーの3〜4倍 |
| いいね判定 | ±340ms | 「だいたい合ってる」を全部拾う |
| ねらったけど外れ | ±550ms | 「おしい!」で努力を認める |
| 関係ないタップ | 減点なし | 連打しても楽しい音が鳴るだけ |
| テンポ | 100〜112 BPM | 童謡のテンポ帯 |
| プレイヤーの拍 | 小節内で固定 | 「3拍目」「4拍目」など毎回同じ場所 |

## 画面遷移

```
boot(音のアンロック)→ menu(3Dステージを周回するカメラ)
  → game(ミニゲーム)→ result(★とファンファーレ)→ menu / retry
```

## リズム同期の仕組み

- 基準クロックは `AudioContext.currentTime`。`performance.now()` は使わない
  (音とのズレが出ないこと最優先)
- 楽曲はステップシーケンサ譜面(16分×16/小節)を **先読みスケジューラ**
  (25ms間隔・180ms先まで)で `OscillatorNode` 等に予約
- `playSong()` が返す開始時刻 `t0` から `拍n の時刻 = t0 + n × (60/BPM)` を全キューに事前計算
- お手本音・演出音も同じ仕組みで 250ms 先まで予約 → 端末負荷で映像が落ちても音は正確
- タップ判定は `audio.now` と最寄り未判定キューの差分で判定

## 3D の絵作り(Switch 風の要素)

| 要素 | 実装 |
| --- | --- |
| セルルック | `MeshToonMaterial` + 4段 `gradientMap`(NearestFilter) |
| リムライト | `onBeforeCompile` でフレネル項を注入 |
| 黒フチ | インバーテッドハル(法線方向に3%膨らませた BackSide メッシュ) |
| トーン | `ACESFilmicToneMapping` + `OutputPass`(sRGB) |
| ブルーム | `UnrealBloomPass`(strength 0.38 / threshold 0.82)— 星・月・電飾だけ光る |
| AA | WebGL2 レンダーターゲットの MSAA×4(`samples: 4`) |
| 影 | 2048px PCFSoft シャドウマップ(ステージ周辺のみカバー) |
| 空 | カスタムシェーダのグラデーションドーム+地平線の残照 |
| 星空 | `Points` カスタムシェーダ(点滅位相・十字のきらめき・拍で膨らむ) |
| アニメ原則 | スカッシュ&ストレッチ、予備動作(振りかぶり=合図を兼ねる)、追従(耳・しっぽ) |

パフォーマンス: ジオメトリはプリミティブのみ(最重でも球の20×16分割)、
パーティクルは `InstancedMesh` プール、DPRは2でクランプ。iPad mini 相当で60fpsを想定。

## ミニゲーム仕様

### 🐰 もちつき ぺったん(104 BPM / 18小節)
- 各小節: 拍1・2 = お手本うさぎが杵で「ぽん・ぽん」、拍3 = プレイヤーの「ぺったん!」
- プレイヤーの杵は 1.5拍前から自動で振りかぶる(予備動作が合図)
- 杵の上のリングが縮み、ぴったりで最小に
- 成功: もちが大きくむにゅっ+キラキラ+米粒紙吹雪。ぴったりでカメラシェイク

### 🐱 おほしさま キャッチ(100 BPM / 18小節)
- 各小節の拍1・3に星が届く。星は2拍かけて左右交互の空からベジェ弧で飛来
- 飛び出しと1拍前に「ひゅん」「きらっ」の予告音
- 成功: 星が弾けてねこの頭の周りを回る星が増えていく
- 失敗: 星はそのまま後ろに流れて消える(悲しい音は鳴らさない)

### 🐻 みんなで ジャンプ(112 BPM / 18小節)
- 拍1・2・3 で うさぎ→ねこ→ほしさん が順に跳ぶ(音程も段々上がる)
- 拍4 が自分(くま)の番。足元のリングが光って教えてくれる
- ぴったり: 紙吹雪+歓声+全員バンザイ

## 意図的にやらなかったこと

- 難易度選択・コンボ・スコア数字 … 4歳には「★がふえる」だけで十分
- 外部フォント・音源・テクスチャ … オフライン動作と起動速度を優先
- Service Worker … 静的ホスティング先が未定のためマニフェストのみ(追加は容易)
