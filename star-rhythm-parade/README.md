# ほしぞら リズムパレード ⭐️

おんがくに あわせて タップ! 4さいから あそべる 3Dリズムあそび。

星空の丘に浮かぶ木のステージで、うさぎ・ねこ・くま・ほしさんたちと一緒に
リズムあそびをする、タップひとつで遊べる 3D リズムゲームです。
iPhone / iPad の縦画面・横画面のどちらでも遊べます。

## あそびかた

1. `index.html` をブラウザで開く(またはWebサーバで配信してアクセス)
2. 「タップして はじめる」を押す(音が出ます 🔊)
3. あそびたいゲームのカードを選ぶ
4. **画面のどこでもいいので、リズムに合わせてタップ!**

| ミニゲーム | リズム | あそびかた |
| --- | --- | --- |
| 🐰 もちつき ぺったん | ぽん・ぽん・**ぺったん!** | お手本うさぎが2回ついたら、3拍目にタップ |
| 🐱 おほしさま キャッチ | ながれぼしが とどいたら | 星が光る輪にとどいた瞬間にタップ |
| 🐻 みんなで ジャンプ | じゅんばんに ジャンプ! | なかまが順番に跳んだら、4拍目に自分の番 |

- 判定はとてもやさしめ(±340msまでOK)。失敗しても減点はなく、キャラが可愛くよろけるだけ。
- 上手にできると ★ がたまり、最後にファンファーレと紙吹雪でお祝い。
- 記録(ベストの★)は端末に保存され、メニューに表示されます。

## ローカルでの起動

ES Modules を使っているため、`file://` では動かないブラウザがあります。
簡易サーバでの起動がおすすめです:

```bash
cd star-rhythm-parade
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

iPhone / iPad では Safari で開いて「ホーム画面に追加」すると全画面で遊べます。

## 技術メモ

- **3D描画**: [three.js](https://threejs.org/) r160(`js/vendor/` に同梱・外部CDN不要)
  - トゥーンシェーディング(4段グラデーション+リムライト)
  - インバーテッドハル方式のアウトライン(キャラの黒フチ)
  - ACES トーンマッピング + UnrealBloom + MSAA×4 + PCF ソフトシャドウ
- **音楽**: WebAudio で完全合成(音源ファイルなし)。ステップシーケンサ+先読みスケジューラ
- **タイミング判定**: AudioContext のクロックを基準に拍→時刻を計算(端末差に強い)
- **依存**: ビルド不要・純粋な静的サイト。オフラインでも動作(PWAマニフェスト同梱)

詳しい設計は [docs/DESIGN.md](./docs/DESIGN.md) を参照。
