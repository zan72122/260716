# きらきら☆パーティーアイランド 🏝️⭐

4さいから あそべる、**3D すごろくパーティーゲーム**。
Switch のパーティーゲームのような、まるくて あかるい トゥーン調 3D 世界を
ブラウザだけで動くように、すべてコードで描いています(外部アセット・ライブラリCDNなし)。

iPhone / iPad の **縦画面でも横画面でも** 遊べます。

## しま は 4つの ゾーン(48マス)

| ゾーン | みどころ |
|---|---|
| 🏖️ はまべ | ヤシのき・いりえ・🐢カメさんの遊覧 |
| 🌴 ジャングル | みっしりの森・いせき・🚠ロープウェイ |
| 🌋 かざん | けむりをはく山・ようがん・ゴロゴロ岩 |
| 🎡 ゆうえんち | かんらんしゃ・テント・🎢コースター |

4にんの どうぶつフレンズ(🐰モモ / 🐤ピヨ / 🐧ペン / 🐸ケロ)は
**それぞれ べつのゾーンから スタート**。ラウンドごとに 天気もかわります
(☀️→☁️→🌧️スコール→🌈あめあがり→🌇さいごの しょうぶ!)。

## 遊び方(おとなむけ)

- 静的ファイルのみ。フォルダを HTTP サーバで配信すれば動きます:
  ```bash
  cd kirakira-party-island
  python3 -m http.server 8000
  # → http://localhost:8000 を iPhone/iPad の Safari で開く
  ```
- ホーム画面に追加すると全画面(PWA マニフェスト同梱)。
- 音はタッチした時点で有効になります(iOS の制限)。右上 🔊 でオン/オフ。

### URL パラメータ(おこのみ調整・テスト用)

| パラメータ | 意味 | 例 |
|---|---|---|
| `rounds` | ラウンド数(既定 5) | `?rounds=3` |
| `mgtime` | ミニゲームの秒数(既定 20) | `?mgtime=15` |
| `mg` | ミニゲームを固定 | `?mg=balloon` (`fruits`/`race`/`stars`/`fish`/`cake`/`cart`/`volcano`) |
| `event` | 大イベントを即時再生 | `?event=boss` (`coaster`/`cave`/`turtle`/`ropeway`/`storm`) |

## 技術メモ

- Three.js r160(`vendor/` に同梱)。ES Modules、ビルド不要。
- トゥーン調は `MeshToonMaterial` + 4段グラデーションマップ、キャラは反転ハル輪郭線。
- 島・海・空・キャラ・サイコロの目・地下どうくつまで、全アセットをプロシージャル生成。
- 効果音・BGM は WebAudio でリアルタイム合成。**ゾーンごとに BGM がかわる**。
- 詳細は [docs/DESIGN.md](docs/DESIGN.md) を参照。
