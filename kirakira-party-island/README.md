# きらきら☆パーティーアイランド 🏝️⭐

4さいから あそべる、**3D すごろくパーティーゲーム**。
Switch のパーティーゲームのような、まるくて あかるい トゥーン調 3D 世界を
ブラウザだけで動くように、すべてコードで描いています(外部アセット・ライブラリCDNなし)。

iPhone / iPad の **縦画面でも横画面でも** 遊べます。

## あそびかた

1. すきな どうぶつフレンズを えらぶ(🐰モモ / 🐤ピヨ / 🐧ペン / 🐸ケロ)
2. じぶんの ばんが きたら、大きな **ピンクのサイコロボタンをタップ!**
3. でた目のぶん ぴょんぴょん すすむ
   - 🔵 コインのマス … コインが もらえる
   - 💗 ハートのマス … みんなに コイン
   - 🌈 にじのマス … 4マス ワープ!
   - 🎆 はなびのマス … おそらに はなび
   - ⭐ スターのマス … **スターゲット!**(スターは おひっこしする)
4. まいラウンドの おわりに **ミニゲーム**(かったら コイン!)
   - 🎈 ふうせん パンパン … ふうせんを タッチで わる
   - 🍎 フルーツ キャッチ … ゆびで かごを うごかして キャッチ
   - 🏁 ぴょんぴょん レース … タップれんだで はしる
   - ⭐ きらきら タッチ … でてきた ほしを タッチ
5. 5ラウンドで おしまい。**スターが いちばん おおい子の ゆうしょう!**

まけても だいじょうぶ。さいごは みんなで おまつりです 🎉

## 遊び方(おとなむけ)

- 静的ファイルのみ。フォルダを HTTP サーバで配信すれば動きます:
  ```bash
  cd kirakira-party-island
  python3 -m http.server 8000
  # → http://localhost:8000 を iPhone/iPad の Safari で開く
  ```
- ホーム画面に追加すると全画面(PWA マニフェスト同梱)。
- 音はタッチした時点で有効になります(iOS の制限)。右上 🔊 でオン/オフ。

### URL パラメータ(おこのみ調整)

| パラメータ | 意味 | 例 |
|---|---|---|
| `rounds` | ラウンド数(既定 5) | `?rounds=3` |
| `mgtime` | ミニゲームの秒数(既定 20) | `?mgtime=15` |
| `mg` | ミニゲームを固定 | `?mg=balloon` (`fruits` / `race` / `stars`) |

## 技術メモ

- Three.js r160(`vendor/` に同梱)。ES Modules、ビルド不要。
- トゥーン調は `MeshToonMaterial` + 4段グラデーションマップ、キャラは反転ハル輪郭線。
- 島・海・空・キャラ・サイコロの目まで、全アセットをプロシージャル生成。
- 効果音・BGM は WebAudio でリアルタイム合成(音声ファイルなし)。
- 詳細は [docs/DESIGN.md](docs/DESIGN.md) を参照。
